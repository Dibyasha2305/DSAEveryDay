<h2><a href="https://www.geeksforgeeks.org/problems/recamans-sequence4856/1?page=3&category=Recursion&sortBy=submissions">Recamans sequence</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 14pt;">Given an integer <strong>n</strong>, return the first n elements of <a href="https://www.geeksforgeeks.org/dsa/recamans-sequence/"><strong><span style="color: #236fa1;">Recaman’s sequence</span></strong></a>. It</span><strong><strong> </strong></strong>starts with <strong><strong>0</strong></strong> as the first term. The (n+1)th term is calculated as <strong><strong>previous term - n</strong></strong>&nbsp;(if positive and not already in sequence); otherwise, use <strong><strong>previous term + n</strong></strong>.</p>
<p><span style="font-size: 14pt;">It is recursively defined as below:<br>Let a(n) denote the (n+1)<sup>th</sup> term. (0 being the 1<sup>st</sup> term).<br></span><span style="font-size: 14pt;">a(0) = 0<br>a(n) = a(n-1) - n, if a(n-1) - n &gt; 0 and is not included in the sequence previously else a(n-1) + n.<br></span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> n = 5
<strong>Output:</strong> 0 1 3 6 2
<strong>Explaination:</strong> <br>a(0) = 0,<br>a(1) = a(0)-1 = 0-1 = -1 and -1&lt;0, therefore a(1) = a(0)+1 = 1,<br>a(2) = a(1)-2 = 1-2 = -1 and -1&lt;0, therefore a(2) = a(1)+2 = 3,<br>a(3) = a(2)-3 = 3-3 = 0 but since 0 is already present in the sequence, a(3) = a(2)+3 = 3+3 = 6,<br>a(4) = a(3)-4 = 6-4 = 2.<br>Therefore the first 5 elements of Recaman's sequence will be 0 1 3 6 2.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input:</strong> n = 3
<strong>Output:</strong> 0 1 3
<strong>Explaination:</strong> As seen in example 1, the first three elements will be 0 1 3.</span></pre>
